# Our Memories With Dad ❤️

A private family memory book website.

## Files
- `index.html` — Main website
- `style.css`  — All styles
- `birthday.mp3` — Background music (أغنية عيد ميلادك)

## How to Deploy on GitHub Pages (Free Hosting)

1. Create a new **private** repository on GitHub (e.g. `family-memories`)
2. Upload all 3 files: `index.html`, `style.css`, `birthday.mp3`
3. Go to **Settings → Pages**
4. Under "Branch" select `main` → `/ (root)` → Save
5. Your site will be live at: `https://YOUR_USERNAME.github.io/family-memories`

> Photos are saved in the browser's localStorage automatically — no server needed!
